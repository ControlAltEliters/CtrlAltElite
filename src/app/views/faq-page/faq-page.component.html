How do I join a game with an open seat?
1. Register an account
2. Log in
3. Find the seat
4. Click JOIN EVENT
How do I reserve a table for a tabletop game?

1. Register an account
2. Log in
3. Navigate to the Events page
4. Click CREATE EVENT
5. Fill in the required fields
6. Click CHOOSE TABLE
7. Select an open table
8. Click CONFIRM TABLE
