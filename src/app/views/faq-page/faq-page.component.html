Frequently Asked Questions

How do I join a game with an open seat?
1. Register an account
2. Find the seat
3. Click JOIN GAME
How do I reserve a table for a tabletop game?

1. Register an account
2. Navigate to the Events page
3. Click CREATE EVENT
4. Fill in the required fields
5. Click CHOOSE TABLE
6. Select an open table
7. Click CONFIRM TABLE

What are the prices?
